class models.MPLayer(in_channels, out_channels)¶

Bases: MessagePassing

A message passing layer for a graph neural network (GNN). The nodes are updated as follows:

\[\mathbf{x}_i^{\prime} = \text{MLP}\left( \Sigma_{j \in \mathcal{N}(i)} \,\{\text{MLP} \left((\mathbf{x}_i \, *\, \mathbf{x}_j)\, ||\, \mathbf{e}_{ji}\right)\} \right)\]
Parameters:

  • in_channels (int) – Number of input features for each node.

  • out_channels (int) – Number of output features for each node.

message(v_i, v_j, e, direction, f_j)¶

Constructs messages from node \(j\) to node \(i\) in analogy to \(\phi_{\mathbf{\Theta}}\) for each edge in edge_index. This function can take any argument as input which was initially passed to propagate(). Furthermore, tensors passed to propagate() can be mapped to the respective nodes \(i\) and \(j\) by appending _i or _j to the variable name, .e.g. x_i and x_j.
